# My Project

## Overview
This project is an end-to-end machine learning pipeline, including:
- Data ingestion
- Data cleaning
- Model training
- Model testing with Streamlit dashboard

## Features
- **Dynamic Data Cleaning**: Separate cleaning for training and testing datasets.
- **Model Training**: Logistic regression with metrics like AUC-ROC, precision, recall, etc.
- **Testing and Continuous Inference**: Supports both manual and cron job-based data processing.
- **Visualization**: Graphs for data drift and model drift.
